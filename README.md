# Call Records API
## Description

A REST API that receives call records and calculates monthly bills for a given telephone number.  

## Installation

`git clone https://github.com/jessicapaz/olist-test.git`

`cd olist-test`

`pipenv install`

`pipenv shell`

`python manage.py migrate`

## Run tests

`python manage.py test`
